# <span style="color: #FF5733;">DKR 01: Основы Docker. Введение</span>

В мире разработки, тестирования и выкладки приложений используется инструмент под названием Docker. Он облегчает эти процессы через использование контейнеров.



<span style="color: #33FF6B;">**Docker**</span> – инструмент, упрощающий разработку, тестирование и выкладку приложений в изолированных контейнерах.

<span style="color: #33FF6B;">**Контейнер**</span> - представляет изолированную среду, где работает приложение, запущенное из *образов*.

<span style="color: #33FF6B;">**Образы**</span> – это архив приложения с необходимыми зависимостями: системными и зависимостями приложения (например, Ruby Gems или Node.js modules).

<span style="color: #33FF6B;">**Хостовая система**</span> – это система, на которой запущен контейнер.

## <span style="color: #3399FF;">Основы Виртуализации и Docker</span>

Виртуализация имеет несколько целей:

- Эффективное использование ресурсов
- Обеспечение безопасности
- Упрощение администрирования

Типы виртуализации включают:

- <span style="color: #FF3366;">Полная виртуализация</span> – гостевая система не осознаёт, что она виртуализована, что может быть медленным.
- <span style="color: #FF3366;">Паравиртуализация</span> – гостевая система модифицирована для более быстрой работы.

<span style="color: #FFCC33;">Контейнерная виртуализация</span> отличается от использования гипервизора тем, что используется только одно ядро операционной системы.

### <span style="color: #3399FF;">Docker и Операционная Система</span>

Контейнер использует ядро операционной системы хоста, что делает его быстрее, чем виртуальные машины. Однако это также означает привязку к определенной среде, в случае Docker – к Linux.

Docker использует возможности Linux, включая:
- namespaces для изоляции ресурсов
- cgroups для ограничения ресурсов
- UnionFS для создания слоев файловой системы

### <span style="color: #3399FF;">Архитектура Docker</span>

Docker работает через REST API, но для удобства взаимодействия используется команда docker в командной строке.
